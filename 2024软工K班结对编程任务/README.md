# 学生点名系统 - 轻量版 PWA

这是一个轻量级的学生点名系统，基于PWA (Progressive Web App) 技术构建。

## 🚀 特点

- **超轻量**: 总体积仅几KB，相比Electron版本减少99%+
- **可安装**: 可以像原生应用一样安装到桌面
- **离线支持**: 支持离线使用，无需网络连接
- **跨平台**: 支持Windows、Mac、Linux、Android、iOS
- **完整功能**: 保留所有原有功能（点名、排名、Excel导入导出）

## 📱 使用方法

### 方法1: 双击启动（推荐）
1. 双击 `启动应用.bat` 文件
2. 系统会自动启动本地服务器并打开浏览器
3. 在浏览器中会看到"安装应用"提示，点击安装即可

### 方法2: 手动启动
1. 在此文件夹中启动本地服务器：
   ```bash
   # 使用Python (推荐)
   python -m http.server 8080
   
   # 或使用Node.js
   npx http-server -p 8080
   ```
2. 打开浏览器访问: http://localhost:8080
3. 点击地址栏右侧的安装图标进行安装

### 方法3: 直接打开
- 直接双击 `index.html` 文件
- 注意：某些功能可能需要本地服务器才能正常工作

## 📦 安装为桌面应用

### Chrome/Edge浏览器:
1. 访问应用后，地址栏右侧会出现安装图标 📱
2. 点击安装图标，选择"安装"
3. 应用会被添加到开始菜单和桌面

### 手机端:
1. 用手机浏览器打开应用
2. 选择"添加到主屏幕"
3. 应用会像原生App一样安装

## ✨ 功能特性

### 🎯 智能点名
- 全部抽点：按顺序点名
- 随机抽点：智能加权随机选择
- 随机提问：支持多级评分
- 幸运机制：积分6的倍数有6倍概率

### 📊 数据管理
- Excel导入：支持学生名单导入
- Excel导出：排名数据导出
- 实时统计：积分排名和出勤率
- 数据持久化：自动保存到本地

### 🏫 班级管理
- 多班级支持
- 学生信息管理
- 出勤状态跟踪

## 🔧 技术特性

- **PWA技术**: 现代Web应用标准
- **Service Worker**: 离线缓存和后台同步
- **Web App Manifest**: 原生应用体验
- **响应式设计**: 适配各种屏幕尺寸
- **本地存储**: 数据安全保存在本地

## 🔍 故障排除

### 问题1: 无法安装应用
- 确保使用支持PWA的浏览器
- 必须通过HTTP/HTTPS访问，不能直接打开文件

### 问题2: Excel功能不工作
- 确保通过本地服务器访问
- 检查浏览器是否支持File API

### 问题3: 数据丢失
- 数据保存在浏览器本地存储中
- 清除浏览器数据会导致数据丢失
- 建议定期导出重要数据

## 📞 支持

如有问题，请检查：
1. 浏览器版本是否支持PWA
2. 是否通过HTTP服务器访问
3. 浏览器控制台是否有错误信息
